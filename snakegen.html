What is SnakeGen?

SnakeGen is a neuro-evolution of the snake game, built from the ground up, with just html, css and js. Tensorflow.js is used to simplify the Neural Network definition, but it could easily be replaced as the architecture is fairly simple.

The goal with this project is to learn more about genetic algorithm, and how they can be used to modify a neural network in order to make the agent "learn".

The agent is the snake, and uses multiple inputs to make its decision.
